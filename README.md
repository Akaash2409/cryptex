
### Run Locally

To run **Cryptex** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Akaash2409/cryptex.git
```

Windows:

```bash
git clone https://github.com/Akaash2409/cryptex.git
```
